# URLSharer

Your personal way to share urls between your machines.

## Requirements

* PHP >= 7.0
* Composer

## Get started

```
$ git clone https://github.com/SaitamaSama/URLSharer
$ composer install
$ vendor/bin/aerys -c src/Bootstrap.php
```
